Welcome to learn Git

Commit > writes changes from local folder to local tracking branch>

Examples: commit from local to local master

Push > delivers changes from local tracking branche to remote branche

Examples: push from local master to remote origin master ( master -> master)

fetch > update changes to remote-tracking branch(origin/master) from remote branche (master)

Examples: updates (origin/master from origin master)

merge > remote-tracking branch into local barnch

Examples: Merge origin/master and remote master

Pull - fetch and merge in one operation

Switch / check-out change your local tracking branch to another local branch
